Cinemateca
2024
This project is a rebrand for the Cinemateca Portuguesa, an iconic and institutional Portuguese cinema found in Lisbon. My intentions with this rebrand was to make it more contemporary, and add a more experimental and fun tone to this institution, making it more dynamic and attracting a younger audience. This project contains a stationary package and merchandising with business cards, a diary, a t-shirt. Additionally I made a website entry page layout, a poster for a movie that would be currently airing, as well as a poster for a cycle of movies.
Academic Project, Editorial Design, Branding & Identity
